# Creando un nuevo repositorio

## Repositorio local (en el computador)

1. "git unit" para inicializar repositorio
2. "git add." o "git add <nombre-archivo>" para agregar cambios al área de preparación
3. "git commit -m "mensaje descriptivo"' para guardar (confirmar) los cambios en el historial des repositorio

## Respositorio remoto (en GiHub)

1. "git push origin main" "git push <repositorio-remoto> <rama-para-enviar> Envia los cambos desde el repositorio local al remoto


## Me arrepenti...
1. "git checkout -- <nombre-archivo-a-restaurar> Descarta los cambios hechos en un archivo del directorio de trabajo y lo devuelve al estado exacto del último commit

## Me arrepenti (segunda opción)

1. 'git restore <nombre-archivo>
